Dice Roller

Exemplo: 1d12+3d6+4

O Dice Roller é uma funcionalidade que, dado um comando, executa e retorna um valor. O comando é uma soma de elementos, onde cada elemento ou é um inteiro, ou é um dado. O dado é escrito na forma "xdn", onde x é o número de dados e n é quantos lados o dado tem. Assim, o comando "1d6+2d8+2" significa rolar 1 dado de 6 lados, dois de 8 lados e adicionar uma constante 2.
